Navigation Structure

📊 Dashboard
Overview metrics, needs-attention alerts, recent activity. The advisor's home screen.
👥 Clients
Household list → Client detail (tabs: GRAT Ladder, Annuity Schedule, History, Settings)
🧮 GRAT Modeling
Standalone modeling tool. Pick client, configure params, live projections, generate proposal.
📈 Reports
Cross-client wealth transfer summary, GRAT performance, fee reporting.
💹 7520 Rate Monitor
Current rate, historical chart, advisory signals, rate impact on active GRATs.
⚙️ Settings
Firm profile, default GRAT parameters, notification preferences. Mostly decorative.
